
| ケース | 処理 |
|--------|------|
| EOF 前の未閉じ lang ブロック | ブロック内として扱い続ける（クラッシュしない） |
| lang ブロック内のコードフェンス | code fence 内の `:::` では閉じない |
| 非 lang fenced div | `lang` key がなければ素通り |
| 属性ブロック後の trailing text | Pandoc fenced div として扱わず通常行として保持 |
| NORMAL 状態の裸の `:::` | そのまま出力 |
| 末尾改行の有無 | 入力の末尾改行状態を保存・復元 |
